¿Que son las Propiedades de CSS?


¿Para que sirve el border-top:?

  • El atributo border-top en HTML y CSS se utiliza para definir el borde superior de un elemento. Este borde puede personalizarse en términos de grosor, estilo y color.

¿Para que sirve el border-boton?

  • El border-bottom en CSS, se utiliza para definir el borde inferior de un elemento. Este borde puede ser personalizado en términos de grosor, estilo y color.

¿Para que sirve el Margin?

  • En CSS, la propiedad margin se utiliza para generar espacio alrededor de los elementos, fuera de sus bordes. Esta propiedad es esencial para controlar el diseño y la separación entre los distintos elementos en una página web.

¿Para que sirve el Padding?

  • En CSS, la propiedad padding se utiliza para generar espacio dentro de un elemento, es decir, entre el contenido del elemento y su borde. Esta propiedad es muy útil para controlar la presentación y la disposición del contenido dentro de los elementos.

¿Para que sirve el text-indent?

  • La propiedad text-indent en CSS se utiliza para especificar la cantidad de espacio que se debe insertar antes de la primera línea de texto en un elemento bloque, como un párrafo "p". Esta propiedad es comúnmente usada para crear sangrías al comienzo de los párrafos.

¿Para que sirve el text-wrap?

  • La propiedad word-wrap (ahora conocida como overflow-wrap) se utiliza para controlar cómo se ajustan las palabras dentro de un contenedor, particularmente cuando una palabra es demasiado larga para caber en una línea.

¿Para que sirve el text-stroke?

  • La propiedad text-stroke en CSS se utiliza para aplicar un contorno o borde alrededor de los caracteres del texto. Esto permite añadir un efecto de borde a los caracteres del texto similar al contorno que se aplica a las formas en SVG.
